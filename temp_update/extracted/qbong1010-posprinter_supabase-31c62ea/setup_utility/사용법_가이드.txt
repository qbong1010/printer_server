========================================
        POS 프린터 프로그램 사용법
========================================

📋 프로그램 시작하기
===================
1. 바탕화면의 "POS 프린터" 아이콘을 더블클릭하세요
2. 또는 run_pos_printer.bat 파일을 더블클릭하세요

🖥️ 화면 구성
============
- 주문 목록: 새로 들어온 주문들이 표시됩니다
- 프린터 상태: 프린터 연결 상태를 확인할 수 있습니다
- 설정: 프린터 및 프로그램 설정을 변경할 수 있습니다

📋 주문 확인하기
===============
- 새 주문이 들어오면 자동으로 화면에 표시됩니다
- 주문 번호, 고객명, 메뉴, 수량, 가격을 확인할 수 있습니다
- 주문 상태는 실시간으로 업데이트됩니다

🖨️ 영수증 출력하기
=================
- 새 주문이 들어오면 자동으로 영수증이 출력됩니다
- 수동으로 출력하려면 주문을 선택하고 "출력" 버튼을 클릭하세요
- 프린터가 연결되어 있지 않으면 오류 메시지가 표시됩니다

⚙️ 설정 변경하기
===============
1. "설정" 탭을 클릭하세요
2. 프린터 설정:
   - 프린터 종류 선택 (USB/네트워크)
   - 프린터 이름 입력
   - 용지 너비 설정
3. "저장" 버튼을 클릭하여 설정을 저장하세요

🔧 문제 해결
===========
프로그램이 실행되지 않는 경우:
1. 관리자 권한으로 실행해보세요
2. 프린터가 연결되어 있는지 확인하세요
3. 인터넷 연결을 확인하세요

프린터가 인식되지 않는 경우:
1. 프린터 전원이 켜져 있는지 확인하세요
2. USB 케이블이 제대로 연결되어 있는지 확인하세요
3. 프린터 드라이버가 설치되어 있는지 확인하세요

주문이 수신되지 않는 경우:
1. 인터넷 연결을 확인하세요
2. Supabase 설정이 올바른지 확인하세요
3. 프로그램을 재시작해보세요

📞 지원 문의
===========
문제가 발생하면 다음 정보를 준비하여 문의하세요:
- 오류 메시지 스크린샷
- app.log 파일 내용
- 프린터 모델명
- Windows 버전

========================================
        프로그램 종료하기
========================================
- 창의 X 버튼을 클릭하세요
- 또는 Alt+F4를 누르세요

주의: 프로그램을 종료하면 주문 수신이 중단됩니다.
      영업 중에는 프로그램을 계속 실행하세요.
